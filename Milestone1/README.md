# Milestone 1 - User Authentication Module

## Project Overview

This project is developed as part of the Infosys Springboard Internship 7.0.

The application is built using Streamlit and provides a complete user authentication system with secure login, signup, forgot password, JWT-based session management, email OTP verification, and an admin dashboard.

## Features

- User Signup
- User Login
- Forgot Password using Security Question
- Forgot Password using Email OTP
- JWT Authentication
- User Dashboard
- Admin Dashboard
- Input Validation

## Technologies Used

- Python
- Streamlit
- JWT (JSON Web Token)
- Gmail SMTP
- Google Colab
- ngrok

## How to Run

1. Open the notebook in Google Colab.
2. Configure Colab Secrets.
3. Run all notebook cells.
4. Open the generated Streamlit URL.
